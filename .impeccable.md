## Design Context

### Users
- Home karaoke hosts and guests using phones for queue/settings and TV for playback.
- Context is quick, in-the-moment control during live social sessions.
- Primary job to be done: configure and run karaoke reliably with minimal friction.

### Brand Personality
- Personality: clear, dependable, modern.
- Emotional goal: users should feel confident and in control when changing settings.
- Tone should prioritize direct feedback over decorative effects.

### Aesthetic Direction
- Keep current dark theme with cyan accent language.
- Favor practical clarity: strong contrast, readable labels, explicit state messaging.
- Avoid over-stylized/glow-heavy CTA treatments that reduce legibility.

### Design Principles
1. Feedback must be immediate and unambiguous for all critical actions.
2. Visual hierarchy should emphasize control state and outcomes first.
3. Interactive elements should look trustworthy and consistent before expressive.
4. Motion and accents should support comprehension, not decoration.
5. Preserve mobile-first usability and tap-target clarity.
